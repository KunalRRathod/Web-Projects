This page describes about HTML Tables

To Create an HTML Table we use table attribute. Since HTML creates tables in making Table row and Table Data

To Define table we need to expres it in <table> tag

We use <tr> tag to define table row and <td> tag to define table data

tabel row heading 1 table row heading 2
Table Data 1 Table Data 2

Describe table in best possible manner


<thead> Define table header class (applies on entire group)
<tbody> Group table body content
<tfoot> Group table footer content

This Defines table in the optimised manner for easy understanding

<table>

<thead>
<tr>
<td>Header Row 1</td>
<td>Header Row 2</td>
<td>Header Row 3</td>
</tr>
</thead>

<tbody>
<tr>
<td>Data Row 1</td>
<td>Data Row 2</td>
<td>Data Row 3</td>
</tr>
<tbody>

</table>

The above code will render table in following way


Header Row 1 Header Row 2 Header Row 3
Data Row 1 Data Row 2 Data Row 3
